install ruby
```
sudo apt install ruby-full
sudo gem install jekyll
```

clone jekyll site
```
https://jekyllthemes.io/theme/beautiful-jekyll
```

build site
```
gem install jekyll bundler
bundle exec jekyll serve --host=0.0.0.0
```

In __config.yml
```
# deployment
host: 0.0.0.0
port: 5000
```
